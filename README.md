GiraHelper
==========

Bash script to control a Gira home server

## Features

* Switch array of Gira power outlets on/off
* CLI-based
* Configurable

## Installation

`git clone http://git.frd.mn/iWelt/gira-helper.git`

`cd gira-helper`

`chmod +x install.sh`

`./install.sh`

Last but not least, adjust the configuration file with:

`vi $HOME/.gira.cfg`
